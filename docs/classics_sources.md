# 典籍语料来源说明

当前 MVP 已在 `data/seed/classics_corpus.py` 内置精选名句，覆盖：

- 诗经
- 楚辞
- 论语
- 周易
- 道德经
- 唐诗
- 宋词
- 古文观止
- 尚书、尔雅、说文、集韵、孟子、大学、中庸等补充来源

这些精选语料用于取名解释与测试夹具，后续批量语料导入不应替代它，而应作为更大语料库的高质量人工精选层。

## 批量导入候选来源

| 来源 | 覆盖内容 | 用途 | 注意事项 |
|---|---|---|---|
| chinese-poetry | 唐诗、宋词、诗经、论语等结构化文本 | 批量构建 `classics` 与 `character_classics` | 导入前确认许可证与 attribution |
| 公开经典文本整理 | 周易、道德经、古文观止等 | 补足非诗词经典 | 需记录版本并做篇章校对 |
| 当前 seed | 精选可取名名句 | 高质量引用、测试夹具 | 保持手工维护 |

参考链接：

- chinese-poetry: https://github.com/chinese-poetry/chinese-poetry

## 导入原则

1. 典籍入库按“句”保存，不按整篇保存，便于前端展示和单字反查。
2. 每句必须保留 `book`、`chapter`、`line_text`、`era`、`author`。
3. 导入时自动建立 `character_classics` 反向索引。
4. 反向索引只记录 CJK 汉字，跳过标点和空白。
5. 当前 seed 语料应优先保留，作为精选层参与排序加权。
